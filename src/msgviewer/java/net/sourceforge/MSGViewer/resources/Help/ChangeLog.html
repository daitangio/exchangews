Version 1.7 (2014-03-16)

  • Moved to sourceforge. Switched from german as sourcede language to english.

Version 1.6.1 (2012-12-23)

  • Perform an autoupdate only once per week, since this causes to much traffic on our server, if the autoupdate is done every program startup.
    By using the -noautoupdate command line switch this can be disabled at all.

Version 1.2 (2011-02-16)
